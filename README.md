# Projekt: Tippspiel-App

Tippspiel-App ist eine Webanwendung zur Organisation und Durchführung von Tippgemeinschaften für Sportereignisse. Nutzer können Tipps abgeben, Punkte sammeln und in einer Rangliste verglichen werden.
